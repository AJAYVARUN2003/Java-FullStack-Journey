# Exception Handling Notes

## try
Contains risky code that may throw an exception.

## catch
Handles the exception if it occurs.

## finally
Executes whether an exception occurs or not.

## Multiple catch
Allows different exceptions to be handled separately.

## Important Rules
- Once an exception occurs, the remaining statements in the try block are skipped.
- finally always executes.
- Catch child exceptions before parent exceptions.
